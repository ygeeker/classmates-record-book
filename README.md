# 电子同学录

## 开发

### 准备工作

1. 修改 `.env.template` 文件名为 `.env`
2. 按照注释修改

### 安装

```sh
pnpm i
```

### 启动

```sh
# 开发
pnpm run dev
# 部署
pnpm run build
pnpm run start
```
